Workflow v1

A skeleton framework for modern web development Written by Duncan Kendrick

Features

Gulp scripts

  • SCSS compiling Uses Libsass insead of Node Sass for speed.
  • Auto vendor prefixing Uses the Can I Use to automagically add last 2 versions of vendor prefixes.
  • CSS & JS file concatenating Combines custom JS & plugins in to a single file to minimise requests to server. Ditto for CSS.
  • CSS & JS minifying Creates .min versions for including in production.
  • JS linting Shows JS errors in console using jshint. Prettifies output using jshint-stylish
  • Image compression Automatically minifies PNG, JPEG, GIF and SVG images
  • Creates PNG versions of SVG files Used in conjunction with Old-IE-Fixes, this ensures IE<9 falls back to PNGs with no additional work.
  • Browser Sync Auto reloads web pages and propagates over network for multi device testing.

Bower

Installs front end dependancies with ease. Default install includes...

  • Susy Lightweight CSS grid framework. Just does grids, no extra fluff.
  • Normalize.css As included in HTML5 Boilerplate.
  • Modernizr - dev version Full dev build for development only. Customise your build for production.
  • jQuery By default use the CDN. This provides a fallback.
  • Old-IE-Fixes - fixes/shims for IE < 9 A collection of shims I’ve bundled together to enable feature support in IE.

Susy & Basic CSS

I have created a very basic skeleton file structure as a jumping off point for writing your project CSS. It’s tied in to Susy but can be used independantly or just removed altogether.
